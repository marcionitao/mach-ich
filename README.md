# Mach-ich

## Mach-ich is a Todo application that will allow a user to:
- View a list of todo’s
- Add new todo’s
- View details of a specific todo (we will be setting up a Master / Detail pattern in Ionic)
- Save the todo’s to permanent storage

## Technologies:
- Ionic
